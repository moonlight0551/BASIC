This is a collection of programs written in BASIC that will run under MBASIC on a CP/M machine.

EVIL.BAS - THIS RUNS UNDER MBASIC. IT COMES FROM A BOOK COMPUTER SPACEGAMES. YOU ARE TRYING TO BLOW UP ELRON THE INVISIBLE ALIEN
IN A 10 X 10 GRID. YOU ARE ASKED WHERE TO PUT THE BOMB. X CAN BE 0 TO 9, Y CAN BE 0 TO 9

FIBO.BAS - FIBONACCI NUMBERS -- ORIGINAL PRG FOR APPLE-1 INTEGER BASIC AUTHOR: UNKNOWN -- ADAPTED FOR MBASIC BY P-LAB 2020

PRIMES.BAS - CAlCULATES PRIME NUMBERS TO 1000

ROCK2.BAS - THIS VERSION IS FROM A SCANNED VERSION FROM 101 BASIC COMPUTER GAMES BY DEC AND CLEANED UP BY CHATGPT
LUNAR1 IS AN INTERACTIVE GAME THAT SIMULATES A LUNAR LANDING. LANDING SIMILAR TO THAT OF THE APOLLO PROGRAM.
THERE IS ABSOLUTELY NO CHANCE INVOLVED.

fibo1.bas - This program runs under MBASIC and prints the first 100 Fibonacci numbers.Tested on a RC2014 running CP/M


julia14.bas - This fractal generator run in MBASIC under CP/M and is the 3rd in the CP/M Fractal Collection. 
It generates famous fractals ie the Spiral, Douady Rabbit and the Dragon on the command line.
Written by CHATGPT at the request of David Lochlin. Each pattern has been finetuned to fit nicely within the 80 x 28 retro display. You will prompted to select which pattern you want to generate and how many iterations. 8 is a good number to use and this has been used in the fine tuning process.

mandel12.bas - This fractal generator calculates a Mandelbrot fractal on the command line. This is the first in the CP/M Fractal Collection.
You will be asked for the number of iterations. 8 is a good number to start with.

sierp2.bas - This fractal generator calculated Sierpinski triangles and is the second in the Fractal Collection.
